🗃️ PaperTrail
PaperTrail is a calm, local‑first project and task tracker built for people who plan in weeks and months, not minutes.
It favors:

clarity over cleverness
persistence over pressure
structure without rigidity

PaperTrail is meant to feel like a well‑used notebook:
quiet, capable, and always there when you need it.

✨ Why PaperTrail?
Most task managers optimize for speed, engagement, or urgency.
PaperTrail optimizes for follow‑through.
It’s designed for:

ongoing work that repeats weekly or monthly
real projects that need notes and documentation
weekly reviews instead of daily guilt
a minimal interface that stays out of the way

No streaks. No gamification. No anxiety.

✅ Core Features
Tasks

One‑off and recurring tasks
Inbox for uncategorized work
Projects with custom colors
Priority, due dates, and status tracking
Bulk actions for efficient cleanup

Work‑Week Planning

Calendar‑based planning for the current work week
Drag‑and‑drop task scheduling
Priority‑colored task indicators
Calendar remains visible even when lists are filtered

Recurring Tasks & Monthly Bundles
PaperTrail is especially well‑suited for recurring work.

Daily, weekly, biweekly, and monthly recurrence
Monthly bundle mode

Define multiple checklist items per month
Check or uncheck items directly
Optional notes when needed (Shift‑click)
Completion history tracked per month


Bundle details stay open while you work

Notes

Notes can stand alone or belong to a project
Inline editing directly in the list
Notes surface automatically during Weekly Review

Weekly Review

Dedicated Weekly Review view
Aggregates:

Completed one‑off tasks
Recurring task history
Notes created during the week


Easy copy for status updates or reflection
Optional CSV export


📊 Dashboard & Insights
The dashboard provides a high‑level overview without pressure:

Due Today, This Week, Overdue, and Recurring summaries
Productivity heatmap for recent activity
Trend charts for productivity, sleep, and mood

PaperTrail also generates simple written observations (e.g. relationships between sleep and productivity) to support reflection rather than raw analysis.

🎨 Interface Philosophy

Clean task cards with actions placed deliberately
Expand/collapse controls using consistent caret patterns
Inline meta chips that wrap naturally
No horizontal scrolling
Light, Dark, and Vapor themes
Focus mode for distraction‑free planning


⌨️ Keyboard Friendly

Quick task entry
Inline editing
Escape to dismiss editors
Shift‑click for optional notes on bundle items


🔐 Privacy & Storage
PaperTrail is local‑first by design.

All data is stored locally using IndexedDB
No accounts required
No telemetry or analytics
Full export and import via plain .json files

You own your data.

📦 Getting Started
Local / Web

Open index.html in a modern browser
(or serve via a local static server for best results)

Desktop (Electron)

Launch the packaged app
All data remains local to your machine


💾 Backups

Use Export Backup from Settings
Import restores projects, tasks, notes, and settings

Backups are human‑readable .json files.
